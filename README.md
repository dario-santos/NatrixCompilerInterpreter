# NatrixCompiler
A compiler for the Natrix language, the final project of the Language Processing course at the University of Beira Interior (UBI).

## How to Test
run the makefile that is in the Ocaml folder and do:

```S
./natrix ./file_to_test
```

or

```S
./natrix < file_to_test
```
